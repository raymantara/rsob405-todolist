# To-Do List App

Приложение для ведения задач.
Ссылка на продукт: https://rsob405-todolistnew.vercel.app/

## Функции
- Добавление задач
- Отметка выполненных (клик по задаче)
- Фильтрация: Все / Активные / Выполненные
- Данные сохраняются в `localStorage`

##  Технологии
- **Frontend**: React + Vite
- **Язык**: JavaScript
- **Стили**: CSS
- **Инструменты**: ESLint, Prettier, GitHub Actions
- **Деплой**: Vercel

## Установка и запуск
git clone https://github.com/raymantara/rsob405-todolist.git
