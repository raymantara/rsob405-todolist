тест-план для To-Do List

1. Добавление задачи
- [ ] Открыть приложение
- [ ] Ввести текст в поле ввода
- [ ] Нажать "Добавить"
- [ ] Убедиться, что задача появилась в списке

2. Сохранение после перезагрузки
- [ ] Добавить задачу
- [ ] Обновить страницу (F5)
- [ ] Убедиться, что задача осталась

3. Отметка выполнения
- [ ] Добавить задачу
- [ ] Кликнуть по ней
- [ ] Убедиться, что текст зачёркнут
- [ ] Кликнуть снова → зачёркивание исчезает

4. Фильтрация задач
- [ ] Добавить 3 задачи
- [ ] Отметить 1 как выполненную
- [ ] Нажать "Активные" → видны только 2 невыполненные
- [ ] Нажать "Выполненные" → видна 1 зачёркнутая
- [ ] Нажать "Все" → видны все 3

5. Пустой ввод
- [ ] Оставить поле пустым
- [ ] Нажать "Добавить"
- [ ] Убедиться, что ничего не добавилось
